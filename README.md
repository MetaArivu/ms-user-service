# User Micro Service

## BUILD & Run Locally
- mvn clean install
- mvn spring-boot:run 
  - This will start server on port 6061

## Build & RUN Docker Image
- mvn package docker:build 
  - This will Build Docker Image
- docker images
  - This will list Docker Images
- Run Docker With Specific Spring Profile ( This Will Go as Env)
  - docker run -p 6061:6061 -e "SPRING_PROFILES_ACTIVE=prod" metamagic/user-service:1.0


## API

- Welcome API
  - curl --location --request GET 'http://localhost:6061/user-service/welcome'

- Register User
  - curl --location --request POST 'http://localhost:6061/user-service/api/v1/register' \
--header 'Content-Type: application/json' \
--data-raw '{
